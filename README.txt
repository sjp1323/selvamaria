
# Restaurante Selvamaría – Sitio web estático

## Estructura
- `index.html`: página principal.
- `styles.css`: estilos globales.
- `app.js`: lógica para cargar y renderizar el menú.
- `data/menu.json`: aquí editas los platos y precios.
- `assets/`: imágenes usadas en la galería.

## Edición rápida del menú
1. Abre `data/menu.json` y cambia `price_k` o agrega nuevos elementos.
2. Guarda y recarga la página. No necesitas tocar HTML.

## Ejecutar localmente
- Recomendado: extensión "Live Server" de VS Code. Haz clic en "Go Live".

## Publicación rápida
- Sube estos archivos a GitHub y habilita GitHub Pages **(branch main, carpeta root)**.
- O súbelos a Netlify (drag-and-drop de la carpeta).
